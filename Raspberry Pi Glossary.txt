<ul>
<li><strong>A/mA</strong> - Ampere/milliampere, the <a href="https://en.wikipedia.org/wiki/Ampere">base unit of electrical current</a>.</li>
<li><strong>API</strong> - <a href="https://en.wikipedia.org/wiki/Application_programming_interface">Application Programming Interface</a>, a set of subroutine definitions, communication protocols, and tools used for building software.</li>
<li><strong>Arm</strong> - A <a href="https://en.wikipedia.org/wiki/Arm_architecture">series of instruction-set architectures and processor microarchitectures</a>, mostly developed by Arm Holdings based in Cambridge, UK.</li>
<li><strong>Armv6</strong> - The instruction set architecture used on the first Raspberry Pi (Pi 1) and the Pi Zero range.</li>
<li><strong>Armv7</strong> - The instruction set architecture used on the Raspberry Pi 2 range.</li>
<li><strong>Armv8</strong> - The instruction set architecture used on the Raspberry Pi 3 range; very similar to Armv7 in 32-bit mode.</li>
<li><strong>balenaEtcher</strong> - A graphical tool for <a href="../installation/installing-images/README.md">programming SD cards</a> ready for use on our devices; available for Linux, Windows, and Mac. Now superceded by Raspberry Pi's own imaging tool, which can be downloaded <a href="https://www.raspberrypi.org/downloads/">here</a>.</li>
<li><strong>Bare metal</strong> - Programming the Raspberry Pi without using an operating system.</li>
<li><strong>BCM2835</strong> - The SoC used on the Raspberry Pi 1, Compute Module 1, and Raspberry Pi Zero range; see our <a href="../hardware/raspberrypi/bcm2835/README.md">official documentation</a>.</li>
<li><strong>BCM2836</strong> - The SoC used on the original Raspberry Pi 2; see our <a href="../hardware/raspberrypi/bcm2836/README.md">official documentation</a>.</li>
<li><strong>BCM2837</strong> - The SoC used on the Raspberry Pi 3, Compute Module 3, and Raspberry Pi 2 version 1.2; see our <a href="../hardware/raspberrypi/bcm2837/README.md">official documentation</a>.</li>
<li><strong>BCM2711</strong> - The SoC used on the Raspberry Pi 4B; see our <a href="../hardware/raspberrypi/bcm2711/README.md">official documentation</a>.</li>
<li><strong>BLE/BTLE</strong> - Bluetooth Low Energy, a low-power version of the Bluetooth wireless communications protocol.</li>
<li><strong>BT</strong> - Bluetooth, a short-range, low-bandwidth, wireless communication protocol often used for mobile devices.</li>
<li><strong>CODEC</strong> - Coder/Decoder, hardware or software blocks that encode and/or decode video or audio data.</li>
<li><strong>Compute Module</strong> - A <a href="../hardware/computemodule/README.md">Pi variant aimed at commercial customers</a>; uses a different form factor, facilitating use inside other products.</li>
<li><strong><code>config.txt</code></strong> - A file located on the boot partition of a Raspberry Pi's SD card that is parsed at boot time in order to enable, disable, and manage operating system features which cannot be changed after booting; see our <a href="../configuration/config-txt/README.md">official documentation</a>.</li>
<li><strong>CSI</strong> - Camera Serial Interface, a hardware interface for connecting cameras to SoCs.</li>
<li><strong>DHCP</strong> - Dynamic Host Configuration Protocol, a network management protocol which dynamically assigns an IP address and other network configuration parameters to each device on a network.</li>
<li><strong>DPI</strong> - Parallel Display Interface, an up to 24-bit parallel RGB interface via the Raspberry Pi's GPIO pins.</li>
<li><strong>DSI</strong> - Display Serial Interface, a hardware interface for connecting LCD panels.</li>
<li><strong>Device Tree (DT)</strong> - A <a href="https://en.wikipedia.org/wiki/Device_tree">data structure for defining hardware</a>.</li>
<li><strong>Device Tree source (DTS)</strong> - A human-readable source file to be compiled into a DTB.</li>
<li><strong>Device Tree blob (DTB)</strong> - A binary file containing a Device Tree; the result of compiling DTS files.</li>
<li><strong><code>dt-blob.bin</code></strong> - A binary file containing initial GPIO configuration that is read by the firmware at boot time; despite the name, this is <strong>not</strong> Device Tree blob (although it is compiled by the Device Tree tools).</li>
<li><strong>DVI</strong> - <a href="https://en.wikipedia.org/wiki/Digital_Visual_Interface">Digital Visual Interface</a>, a video interface for connecting a source device to a monitor or display; electrically compatible with HDMI, so a simple adapter can convert from one to the other (excluding audio, which DVI does not support).</li>
<li><strong>Etcher</strong> - See balenaEtcher.</li>
<li><strong>FAQ</strong> - Frequently Asked Questions! See our <a href="../faqs/">FAQ</a>.</li>
<li><strong>Firmware</strong> - Software that runs on a particular hardware device, for example a wireless chip or dongle; distinct from the usual software running on the Raspberry Pi in that it is usually a fixed binary blob of data that is loaded on to the device at startup; on the Raspberry Pi, the VC4 GPU requires firmware to be loaded to it at startup in order boot up and run the entire system.</li>
<li><strong>GNU</strong> - The <a href="https://en.wikipedia.org/wiki/GNU_Project">GNU Project</a> is a free software mass collaboration project, which has produced huge amounts of software that is used on the Raspberry Pi, including Linux itself.</li>
<li><strong>GPIO</strong> - General Purpose Input/Output, the programmable pins on the Raspberry Pi.</li>
<li><strong>GPU</strong> - Graphical Processing Unit, a hardware device for handling graphical (and related) tasks at high speed; on the Raspberry Pi range up to the Pi 3B+, this is called the VideoCore4 (VC4), and is built into the SoC. On the Pi 4, a newer and faster version called VideoCore6 (VC6) is used.</li>
<li><strong>HAT</strong> - Hardware Attached on Top, a specification for designing devices to attach to the Raspberry Pi; see the <a href="https://www.raspberrypi.org/blog/introducing-raspberry-pi-hats/">launch announcement</a>.</li>
<li><strong>HDMI</strong> - <a href="https://en.wikipedia.org/wiki/HDMI">High-Definition Multimedia Interface</a>, a standard interface for transmitting uncompressed video and compressed or uncompressed audio data from a source device to a display device.</li>
<li><strong>HDCP</strong> - <a href="https://en.wikipedia.org/wiki/High-bandwidth_Digital_Content_Protection">High-bandwidth Digital Content Protection</a>, an optional encryption mechanism used for HDMI or DVI transmission.</li>
<li><strong>HVS</strong> - Hardware Video Scaler, a hardware block in the VC4 or VC6 GPU on all Raspberry Pi models that is used for manipulating and displaying bitmaps.</li>
<li><strong>I<sup>2</sup>C/I2C/IIC</strong> - <a href="https://en.wikipedia.org/wiki/I%C2%B2C">Inter-Integrated Circuit</a> (pronounced <em>I-squared-C</em>), an electrical protocol used for attaching lower-speed peripheral ICs to processors and microcontrollers in short-distance, intra-board communication.</li>
<li><strong>I<sup>2</sup>S</strong> - <a href="https://en.wikipedia.org/wiki/I%C2%B2S">Inter-Integrated Circuit Sound</a>, an electrical serial bus interface standard used for connecting digital audio devices together.</li>
<li><strong>ISP</strong> - Imaging System Pipeline, a set of hardware (and sometimes software) stages that processes images from a camera into a high-quality result; the Raspberry Pi has a built-in hardware ISP that processes images captured from a camera attached to the CSI port.</li>
<li><strong>ISP</strong> - Internet service provider.</li>
<li><strong>LAN</strong> - Local Area Network, an Ethernet- or wireless-based communication network in a localised area, for example a house or office.</li>
<li><strong>LED</strong> - Light-Emitting Diode, a semiconductor device that produces light using electricity.</li>
<li><strong>Linux</strong> - The main operating system used on the Raspberry Pi; the particular customised distribution for the Raspberry Pi range is called Raspbian; third parties have also produced their own distributions.</li>
<li><strong>MSD</strong> - Mass Storage Device (amongst other things), such as an SD card or hard disk drive.</li>
<li><strong>MQTT</strong> - MQTT stands for MQ Telemetry Transport, a lightweight and simple machine-to-machine protocol often used for IoT devices; see <a href="http://mqtt.org/">the MQTT website</a>.</li>
<li><strong>NAND</strong> - A <a href="https://en.wikipedia.org/wiki/NAND_gate">NAND gate</a> is a logic gate, specifically an inverted AND gate; can also refer to a type of flash memory that uses NAND logic.</li>
<li><strong>NOOBS</strong> - New Out of Box software, easy <a href="../installation/noobs.md">OS system image installation</a> for the Raspberry Pi range; <strong>note:</strong> NOOBS is no longer developed by the Raspberry Pi team, see PINN instead.</li>
<li><strong>OTP</strong> - One-Time Programmable, a type of memory that can only be programmed once, and retains the programmed data after power off; used for serial numbers etc.</li>
<li><strong>Overlay</strong> - A boot-time or run-time patch to a Device Tree allowing a degree of configuration by the user; see our <a href="../configuration/device-tree.md">official documentation</a> for more details, or run <code>dtoverlay -a</code> in a terminal window for a list of available overlays.</li>
<li><strong>Pink Pony/Unicorn</strong> - A feature request for a future Raspberry Pi model that is unlikely to happen.</li>
<li><strong>PINN</strong> - A <a href="https://github.com/procount/pinn">third-party development</a> of the NOOBS system, often a better option than the official NOOBS.</li>
<li><strong>PoE</strong> - Power over Ethernet, a mechanism to power a device through its Ethernet connection; we produce a <a href="https://www.raspberrypi.org/products/poe-hat">PoE HAT</a> for the Raspberry Pi 3 Model B+ and Raspberry Pi 4 Model B.</li>
<li><strong>Polyfuse</strong> - A self-resetting electrical fuse used on some Pi models to protect against over-current; can take hours or even days to reset.</li>
<li><strong><code>raspi-config</code></strong> - A command line tool for configuring Raspbian's operating system features; see our <a href="../configuration/raspi-config.md">official documentation</a>.</li>
<li><strong>Raspberry Pi Imager</strong> - The official graphical tool for downloading and installing operating system images to an SD card. Download it <a href="https://www.raspberrypi.org/downloads/">here</a>.</li>
<li><strong>raspivid, raspistill, raspiyuv, raspividyuv</strong> - A set of applications to run the Raspberry Pi Camera Module in various modes; see our <a href="../raspbian/applications/camera.md">official documentation</a>.</li>
<li><strong>Raspbian</strong> - A Debian-based Linux distribution targeted at the entire Raspberry Pi range, regarded as the official operating system; the current version is based on Debian Buster, with the first release in June 2019; see the <a href="https://en.wikipedia.org/wiki/Raspbian">Wikipedia entry for further release details</a>.</li>
<li><strong>Raspberry Pi Configuration Tool (<code>rcgui</code>)</strong> - A graphical equivalent of <code>raspi-config</code>.</li>
<li><strong>Raspberry Pi Desktop</strong> - A version of the LXDE desktop environment, specifically tailored to the Raspberry Pi (used to be called PIXEL).</li>
<li><strong>RPF</strong> - Raspberry Pi Foundation, the original, UK-registered educational charity (1129409) that launched the Raspberry Pi.</li>
<li><strong>RPT/RPF(T)</strong> - Raspberry Pi Trading, a wholely owned commercial subsidiary of the RPF which designs and develops all Raspberry Pi products, including magazines; all profits from the RPT go to the RPF for use in their educational programmes.</li>
<li><strong>rpi-update</strong> - A program you can run on a Raspberry Pi to download the very latest test firmware and Linux kernel; there is no guarantee that the latest version will work correctly, so you should only run rpi-update if recommended by a Raspberry Pi engineer.</li>
<li><strong>SoC</strong> - System on a Chip, an integrated circuit that comprises all components of a computer.</li>
<li><strong>SPI</strong> - Serial Peripheral Interface bus, a synchronous serial communication interface specification used for short distance communication.</li>
<li><strong>SSH</strong> - Secure Shell, a cryptographic network protocol for operating network services securely over an unsecured network; see our <a href="../remote-access/ssh/README.md">official documentation</a> or <a href="https://en.wikipedia.org/wiki/Secure_Shell">the Wikipedia article on SSH</a>.</li>
<li><strong>TLA</strong> - Three-letter abbreviation, almost the entire reason for this glossary.</li>
<li><strong>TLS</strong> - <a href="https://en.wikipedia.org/wiki/Transport_Layer_Security">Transport Layer Security</a>, a cryptographic protocol used to supply security over a computer network, often used to secure other protocols such as MQTT.</li>
<li><strong>TP1/TP2</strong> - Test points.</li>
<li><strong>UART</strong> - Universal Asynchronous Receiver-Transmitter, a protocol for asynchronous serial communication in which the data format and transmission speeds are configurable.</li>
<li><strong>USB</strong> - <a href="https://en.wikipedia.org/wiki/USB">Universal Serial Bus</a>, the main Raspberry Pi models have four USB sockets for connecting USB devices, for example mice or keyboards.</li>
<li><strong>VC4, VC6</strong> - The VideoCore4 or VideoCore6 GPU used on the Raspberry Pi; contains a large number of hardware blocks that handle graphics, cameras, display, CODECs, etc.</li>
<li><strong>VNC</strong> - A graphical desktop sharing system that uses the Remote Frame Buffer protocol (RFB) to remotely control another computer; see our <a href="../remote-access/vnc/README.md">official documentation</a>.</li>
<li><strong>Volt (V)</strong> - SI-derived unit for <a href="https://en.wikipedia.org/wiki/Volt">electrical potential difference</a>.</li>
<li><strong><code>vcgencmd</code></strong> - A Raspberry Pi-specific tool for communicating with the VideoCore GPU.</li>
<li><strong>Watt</strong> - A measure of <a href="https://en.wikipedia.org/wiki/Watt">power</a> often used when specifying power supplies.</li>
</ul>
